# Chiros Swarm

Repo with all bat inside every sub folder !
Object are run with a simple flask app.

__Structure__

repo <br>
  | <br>
  -- object1 <br>
  -- object2
  ...

__Utilization__

*SetUp*
```
cd object1
python3 -m venv .
source ./bin/activate
pip install -r requirements.txt
```

*Quit virtualenv*
`deactivate`

*Utilization*
```
python your_file.py
```
